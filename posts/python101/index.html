My idea of audience will be of new students who wanna try python or a science students who starting to learn python or tinker who wanna automate daily stuff in python.

Python is a high level programming language, designed by Guido van rossum. It's main objective is code readability. First appeared in 1991, it is one of the leading languages in 2022. It is used in web developing, in data science and analysis, in scietific computing etc. It has reputation for being so slow than other high level programming languages like C, C++, java, Rust etc. Main reason is how it got executed.
